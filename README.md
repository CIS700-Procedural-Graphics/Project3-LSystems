##Axiom and Grammar

I used the axiom:

```
FX
```
<br><br>
Where, <br>

```
X --> FX[+F--FX][-F++FX][UFVVFX][VFUUFX]
```
<br><br>
And <br>
```
U
```
and 
```
V
```
rotate about the z axis + and - respectively. <br><br>

The user can manipulate the angle of the buds
<br><br>
